Release Notes

v0.3 (2nd December 2012)

  • Bug fix #13
    The validation kept validating results when the result format was invalid, leading to misleading validation errors. This is now fixed.
  • Bug fix #17
    The Sheets contents pane sometimes didn't show the best and average results. This is now fixed.
  • Bug fix #22
    The recognition of the result format for multiple blindfolded was sometimes incorrect if the round had only DNF results. This is now fixed.
  • Bug fix #25
    Workbooks that contained cell formats with Locale information didn't always open properly. This is now fixed.
  • Improvement #6
    The Persons query now only updates persons who competed in the uploaded competition.
  • Improvement #27
    The Persons query now also returns the persons who still have a missing date of birth or gender.

v0.2 (29th October 2012)

  • Bug fix #2
    The workbook file used to be locked by the application under Windows, which prevented making changes using Excel and refreshing from the application. This is now fixed.
  • Bug fix #5
    The validation didn't check that non-empty results are formatted correctly for combined rounds. This is now fixed.
  • Improvement #3
    The validation now also checks for wrongly calculated best and average results.
  • Improvement #4
    The validation now also checks for rounding of times >10 minutes.
