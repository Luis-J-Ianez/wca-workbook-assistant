Release Notes

v0.2 (29th October 2012)

  • Bug fix [WCAUPLOADER-1]
    The workbook file used to be locked by the application under Windows, which prevented making changes using Excel and refreshing from the application. This is now fixed.
  • Bug fix [WCAUPLOADER-15]
    The validation didn't check that non-empty results are formatted correctly for combined rounds. This is now fixed.
  • Improvement [WCAUPLOADER-2]
    The validation now also checks for wrongly calculated best and average results.
  • Improvement [WCAUPLOADER-3]
    The validation now also checks for rounding of times >10 minutes.
